1	Summary	После удаления товара из корзины, число товара в корзине не меняется.					
	Description	Шаги воспроизведения:					
		1. Зайти на сайт https://aliexpress.ru/ зарегестрированым пользователем.					
		2. Добавить 3 любых товара в корзину.					
		3. Перейти в корзину.					
		4. Удалить 1 товар из корзины.					
		5. Подтвердить удаление.					
	Expected results	После удаления товара из корзины, число товаров меняется на актуальное значение.					
	Actual results	Число товара в корзине остается не изменным.					
	Priority	Low.					
	Severity	Trivial.					
	Environment	Google Chrome Версия 103.0.5060.134 (Официальная сборка), (64 бит)

	Description	Шаги воспроизведения:					
		1. Зайти на сайт https://aliexpress.ru/ зарегестрированым пользователем.					
		2. Добавить в корзину товар с платной доставкой.

	Actual results	Сумма указана без учета доставки.					
	Priority 	Low.					
	Severity	Trivial.					
	Environment	Google Chrome Версия 103.0.5060.134 (Официальная сборка), (64 бит)
